Requirements:
Windows XP/Vista/7
OpenGL 3.0 capable graphics hardware
128mb available memory

Controls:
+/-          Increase / Decrease animation speed
R            Reset animation speed
Cursor Keys  Rotate snowglobe
S            Toggle sun/spot lighting
M            Cycle tree render modes
F9           Toggle fullscreen/windowed
